  <h1>Personal Notes Manager</h1>
  <p>A full-stack web application for managing personal notes. Features include adding, editing, deleting, filtering, and sorting notes with a clean and user-friendly interface.</p>

  <h2>Features</h2>
  <ul>
    <li>Create, edit, and delete notes.</li>
    <li>Mark notes as completed or not completed with color toggling.</li>
    <li>Filter and search notes by title.</li>
    <li>Sorted notes by the latest creation date.</li>
    <li>Persistent storage with SQLite database.</li>
  </ul>

  <h2>Installation and Setup</h2>
  <ol>
    <li>Clone the repository</li>
    <li>Navigate to the backend directory and install dependencies</li>
    <li>Start the backend server</li>
    <li>Navigate to the frontend directory and install dependencies</li>
    <li>Start the frontend application:
      <pre><code>npm start</code></pre>
    </li>
  </ol>

  <h2>Usage</h2>
  <p>After starting both the backend and frontend, open the application in your browser at <code>http://localhost:3000</code>.</p>
